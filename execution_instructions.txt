



Compilation instructions:
1. cd into the required directory (tcp, udp, rmi)
2. Compile files using javac Program.java

Running instructions:
a. For UDP:
Running command syntax (For 2 nodes. Can be extended similarly to n nodes):
java Program <current_node_id> <total_node_count> <node_1_id>:<node_1_ip_address> <node_2_id>:<node_2_ip_address>

Example commands for running 2, 3, 4 nodes:
3. For running 2 nodes run these commands in 2 different terminals:
    For node 0:
    a. java Program 0 2 0:localhost 1:localhost 

    For node 1:
    b. java Program 1 2 0:localhost 1:localhost 

4. Similarly For running 3 nodes run these commands in 3 different terminals:
    For node 0:
    a. java Program 0 3 0:localhost 1:localhost 2:localhost

    For node 1:
    b. java Program 1 3 0:localhost 1:localhost 2:localhost

    For node 2:
    c. java Program 2 3 0:localhost 1:localhost 2:localhost


5. For running 4 nodes run these commands in 4 different terminals:
    For node 0:
    a. java Program 0 4 0:localhost 1:localhost 2:localhost 3:localhost

    For node 1:
    b. java Program 1 4 0:localhost 1:localhost 2:localhost 3:localhost

    For node 2:
    c. java Program 2 4 0:localhost 1:localhost 2:localhost 3:localhost

    For node 3:
    d. java Program 3 4 0:localhost 1:localhost 2:localhost 3:localhost
